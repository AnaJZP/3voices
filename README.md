# Three Voices of Sustainability

**A Computational Discourse Analysis of Academic, Institutional, and Public Narratives (2015–2025)**

[![Python 3.12+](https://img.shields.io/badge/Python-3.12%2B-blue.svg)](https://www.python.org/)

---

## Overview

This repository contains the complete analysis pipeline and results for *"Three Voices of Sustainability"*, a study comparing how academic researchers, governmental institutions, and the general public discuss sustainability. The analysis uses:

- **BERTopic** neural topic modeling for thematic discovery
- **Transformer-based multilingual sentiment analysis** for evaluative stance
- **Semantic co-occurrence network analysis** for conceptual structure

The corpus comprises **15,233 documents** (2015–2025): 11,741 academic abstracts (OpenAlex), 282 institutional documents (UK Government, WEF, UNEP, UN Secretary-General), and 3,210 YouTube comments on sustainability-themed videos.

## Key Findings

| Finding | Detail |
|---|---|
| **Sentiment gap** | Institutional discourse is most positive (*M* = 0.81); public commentary most skeptical (*M* = 0.53); academic in between (*M* = 0.57) |
| **Academic decline** | Academic sentiment declined ~20% from 2015 to 2025, with a change point in 2021 |
| **Discursive silences** | Circular economy, waste valorization, and AI — prominent in academia — are absent from public discourse |
| **Network structure** | Academic networks are modular (*Q* = 0.42); public networks are diffuse (*Q* = 0.11); institutional networks are dense but conceptually shallow |

## Installation

### Prerequisites

- Python 3.12+
- GPU recommended for BERTopic embedding generation (CPU works but is slower)

### Setup

```bash
# Clone the repository
git clone https://github.com/AnaJZP/3voices.git
cd 3voices

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
# or: venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt
```

## Usage

### Full Pipeline

```bash
# Run the complete pipeline (collect → preprocess → analyze)
python run_full_pipeline.py
```

### Individual Steps

```bash
# Step 1: Collect data from OpenAlex, YouTube, and institutional sources
python main_collect.py

# Step 2: Preprocess and clean texts
python main_preprocess.py

# Step 3: Run BERTopic, sentiment analysis, and network analysis
python main_analyze.py
```

### Regenerate Figures

```bash
# Regenerate all figures from saved results (no recomputation needed)
python fix_figures.py
```

## Methods Summary

### Topic Modeling
- **Model**: BERTopic (Grootendorst, 2022)
- **Embeddings**: `paraphrase-multilingual-MiniLM-L12-v2` (384-dim)
- **Dimensionality reduction**: UMAP (*n_neighbors* = 15, *min_dist* = 0.0)
- **Clustering**: HDBSCAN (*min_cluster_size* = 50)
- **Two models**: Academic-only (21 topics) and combined (11 topics)

### Sentiment Analysis
- **English**: `nlptown/bert-base-multilingual-uncased-sentiment`
- **Spanish**: `pysentimiento/robertuito-sentiment-analysis`
- **Multilingual fallback**: `cardiffnlp/twitter-xlm-roberta-base-sentiment-multilingual`
- **Statistical tests**: Kruskal–Wallis, Dunn's post-hoc (Bonferroni), Cliff's delta

### Semantic Networks
- Weighted co-occurrence within a 5-token sliding window
- Top 100 content words per subcorpus
- Metrics: density, weighted clustering (Onnela), modularity (Louvain), centrality measures

## Data Availability

| Source | Access | Notes |
|---|---|---|
| Academic abstracts | [OpenAlex](https://openalex.org) | Raw query results included in `data/raw/academic/` |
| Institutional documents | Public APIs | Included in `data/raw/institutional/` |
| YouTube comments | YouTube Data API | Included in `data/raw/public/` (usernames removed) |

> **Note**: Large intermediate files (processed CSVs >50 MB, BERTopic models ~550 MB, embedding caches) are excluded from the repository via `.gitignore`. These files are regenerated by the pipeline.
